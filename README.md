## Ideia incial: 
O Mapa do Hexa, uma aplicação para informar, integrar e criar eventos onde os habitantes de floripa podem assistir os jogos da copa do mundo de 2026.

## Como funcionará:
- Organizadores ( Donos de bares, Centros Academicos e outros grupos ) - Cadastrariam o evento e sua descrição, informando o que existirá no ambiente e o que precisaria ser levado.
- Outros usuários - Visualiza os locais disponíveis e se confirma no evento, para ajudar na contabilização de presença e divulgação do local

## Composição (pode mudar um pouco):
- Lista de eventos ( com possibilidade de mapa ) 
- Contador de Pessoas inscritas no evento
- Possibilidade de compartilhar o evento que o usuário se inscreveu com outros usuários
- Mural de recados ( Avisos do organizador do evento para com os inscritos )
